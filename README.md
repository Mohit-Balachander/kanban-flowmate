## Project Overview

This project delivers a full-stack, production-ready project management application engineered for **observability**. It integrates a modern React-based Kanban board with a complete monitoring suite, demonstrating the seamless fusion of a functional user application with a robust, real-time performance analysis backend.

---

## Core Objective

The primary objective is to provide a comprehensive, out-of-the-box system where **application performance monitoring is a core, integrated feature**, not an afterthought. The project serves as a powerful demonstration of DevOps principles, showcasing how modern web applications can be architected for reliability, transparency, and deep system insight.

---

## Technical Architecture

The system is built on a modular, containerized architecture composed of three main pillars:

- **Frontend Application**: A dynamic and responsive user interface built with **React**, **TypeScript**, and **Vite**.

- **Backend Service**: A lightweight and efficient API server powered by **Node.js** and **Express**, responsible for data persistence and metric exposition.

- **Monitoring Stack**: An integrated observability suite containerized with **Docker**, featuring:
  - **Prometheus**: For time-series data collection and alerting.
  - **Grafana**: For advanced data visualization and dashboarding.
